# Brand Motion Quiz — APK Build

## Cum obții APK-ul în 5 pași

### Pasul 1 — Creează repository-ul pe GitHub
1. Mergi pe github.com și ești logat
2. Click pe **"+"** (dreapta sus) → **"New repository"**
3. Nume: `brand-motion-quiz`
4. Setează **Public**
5. **NU** bifa "Add README"
6. Click **"Create repository"**

### Pasul 2 — Încarci fișierele
Pe pagina noului repository gol, vei vedea opțiunea **"uploading an existing file"**
1. Click pe **"uploading an existing file"**
2. Trage tot conținutul din folderul `quiz-apk` (toate fișierele și folderele)
3. Scroll jos → scrie "Initial commit" → click **"Commit changes"**

### Pasul 3 — Pornești build-ul
1. Click pe tab-ul **"Actions"** din repository
2. Vezi workflow-ul **"Build APK"** → click pe el
3. Click **"Run workflow"** → **"Run workflow"** (buton verde)
4. Aștepți ~5-8 minute

### Pasul 4 — Descarci APK-ul
1. După ce build-ul devine verde ✅, click pe el
2. Scroll jos la **"Artifacts"**
3. Click **"BrandMotionQuiz-APK"** → se descarcă un ZIP
4. Dezarhivezi ZIP-ul → ai fișierul `app-debug.apk`

### Pasul 5 — Instalezi pe Android
1. Trimite APK-ul pe telefon (WhatsApp, email, Google Drive)
2. Pe telefon: **Setări → Securitate → Instalare din surse necunoscute** → Activează
3. Deschide fișierul APK → **Instalează**

---
Aplicația va apărea pe telefon cu numele **"Brand Motion Quiz"** și icoană portocalie.
